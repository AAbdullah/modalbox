Installation

  1. Download ModalBox and unpack to your /includes directory
  2. Include the following JavaScript files in your pages: 
    <script type="text/javascript" src="includes/prototype.js"></script>
<script type="text/javascript" src="includes/scriptaculous.js"></script>
<script type="text/javascript" src="includes/modalbox.js"></script>
  3. Include the CSS file modalbox.css in your pages: 
    <link rel="stylesheet" href="includes/modalbox.css" type="text/css" media="screen" />

Reference

The Modalbox Class

ModalBox is a JS OO-style class. It has several public methods which can be called on your instance of ModalBox object.

Method initialize

This method is being called on New Modalbox(...)

initialize: function(title, url, options)

title — string. Defines the title of the ModalBox window

url — URI. URL to load inside the window

options — set of options.

Options
  • overlayClose: true | false Close modal box by clicking on overlay. Default is true
  • width: integer Width in pixels. Default is 400px
  • height: integer Height in pixels. Default is 400px

Method show

Redraws the ModalBox window with the given parameters

show: function(title, url, options)

Parameters are the same as for initialize method.


Method hide

Hides ModalBox window. No parameters required.

hide: function()

How to use

Calling ModalBox for a first time

Create a link with href and title attibutes filled. We will use them as a parameters in our example:

<a href="frame_send-link.html" title="Simple form" 
onclick="globalMB = new Modalbox(this.title, this.href, {width: 600});
return false;">Send link to a friend process</a>

globalMB is used to save the pointer to our window so we can access it from inside the ModalBox window.

Pop-up creation

To create a simple pop-up (as shown in the Examples), we'll use globalMB. The second frame is a separate HTML file. To call it, we need to do the following:

<button type="button" onclick="globalMB.show('Sending status', 
'frame_send-link-done.html', {width: 460, height:150});">Send link</button>

We're calling method show with new parameters. As you can see, not only anchor elements could be used to call ModalBox.

Hiding ModalBox

To hide a ModalBox window with a link inside it, use the following:

<a href="#" title="Close window" onclick="globalMB.hide(); return false;">Close</a>
